Expand description

Any struct can be used as the fib heap element simply by embedding a RawNode in it (or wrapping it in a struct containing a raw node) and implementing this trait.

fn get_raw(&'a self) -> &RawNode<'a, Self>

Get a reference to the embedded raw node. This is used internally to traverse and bookkeep the heap. Accessing this is not thread safe as-is. Implementors can place the raw node under a lock, but it’s better to lock the entire FibHeap to avoid race conditions.

Object Safety§

This trait is not object safe.
